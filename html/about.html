This site is used to demonstrate the basics of UI-Router for AngularJs. Data for this site come from mysportsfeeds.com

Get the code at https://github.com/wznuk/NBA-Stats-UI-Router

This is the 'about' state. Like the 'stats' state this state does not have a parent. This state, unlike the others does not have an associated component or controller, but rather a simple templateUrl.
