This is the 'stats.team.player.basic-stats' state. This is a child of the 'player' state and a great-grandchild of the 'stats' state. Notice, that we never navigated to the 'player' state. This is because the 'player' state is abstract (abstract: true). An abstract state cannot be navigated to on it's own, but you can navigate to it's children.

This is the 'stats.team.player.extended-stats' state. This is also a child of the 'player' state.

Note that a child state is able to inherit the resolved bindings from it's parent state.' For example, the 'stats.team' state resolves a variable called "team" that contains related team information, including the team's primary color. The 'stats.team.player' state takes advantage of this variable and uses it to dynamically color the line seen in the modal above.
